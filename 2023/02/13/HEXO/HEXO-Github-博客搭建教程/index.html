HEXO + Github 博客搭建教程

搭建教程：https://blog.csdn.net/qq_58608526/article/details/124652412

1.先创建一个空的文件夹，如 LeeBolg

2.创建blog命令

3.启动本地 blog

4.退出本地 blog 后，创建新的文章

创建的文章会生成在 LeeBolg/source/_posts/ 目录下

5.回到 LeeBolg 目录下，清理一下

6.生成一下，刚刚写的博客就已经被添加到本地的博客里面去了

7.重新启动一下博客，刚刚写的博客就可以在本地博客里面看到

8.在github上新建一个仓库，并以用户名.github.io 命名

9.使用cnpm安装一个插件，如果cnpm还未安装就先安装cnpm

10.编辑 LeeBlog 目录下的 _config.yml 文件的 deploment 部分：

11.部署到远端

12.推送成功后，访问https://lyx2099.github.io 就可以看到自己的在线博客了

13.更换主题，以 yilia 主题为例，克隆该主题

14.克隆的主题在 themes 目录下，可以进入该目录下查看是否存在 yilia 主题的目录

15.回到 LeeBlog 目录，修改 _config.yaml 文件，默认主题是 landscape，将其修改为 yilia

16.清理并生成然后启动，启动后可以在本地 http://localhost:4000/ 预览一下

17.使用 heox d 推送到远端，需要一点时间远端才能将主题更新过来
