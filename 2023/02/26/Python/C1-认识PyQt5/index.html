C1 认识PyQt5

PyQt5 框架简介

PyQt5 的特点

  • 基于高性能的 Qt 的 GUI 控件集
  • 跨平台（Windows、Linux、MacOS）
  • 使用 信号/ 槽 （signal/slot）机制进行通信
  • 对 Qt 库完全封装
  • 可以使用 Qt 成熟的 IDE （如 Qt Designer）进行图形界面设计，并自动生成可执行的 Python 代码
  • 提供种类繁多的窗口控件

Qt 与 PyQt 的关系

  • PyQt 是 Qt 框架的 Python 语言实现，每一个 PyQt 控件都有其对应的Qt控件
  • 相比于 Qt，使用 PyQt 进行开发的产品，代码量少，开发效率更高
  • PyQt和 Qt 的设计是完全面向对象的
  • Qt 和 PyQt 使用 信号/槽 机制在窗口控件之间传递事件和消息，相比于其他图形界面开发所采用的回溯（callback）机制，使用信号/槽更安全和简洁

其他图形界面开发库介绍

  • Tkinter
  • wxPython
  • PyGTK
  • PySide

PyQt4 与 PyQt5

  • PyQt5 不向下兼容 PyQt4 编写的程序
  • PyQt5 不支持 Python 2.6 以前的版本，默认支持 python3
  • PyQt5 新式信号/槽使用更简单
  • 新手建议从PyQt5开始学习

PyQt5 环境搭建

PyQt5 环境搭建（windows）

  • 安装Python3：python3安装教程

  • 安装PyQt5

  • 安装图形界面开发工具 Qt Designer

    • 安装：pip install PyQt5-tools-i https://pypi.douban.com/simple

    • 环境配置

      • PyQt5 和 PyQt-tools 安装成功后会在 Python 安装目录（例如我的是：D:\Python39\）的 Lib\Site-packages 目录下看到安装包

      • 把你 pyqt5-tools 所在目录 (例如我的是：D:\Python39\Lib\Site-packages\pyqt5-tools) 添加到系统变量 path 里面

      • 此电脑->右键属性->高级系统设置->环境变量->系统变量path->添加上面的 pyqt5-tools 所在的目录

  • 测试 PyQt5 是否安装成功
